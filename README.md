# 🏙️ Catalyst: Student Ranking Hub

A high-performance Academic Ranking & Performance system designed for Faculty and Students. Built with a "White-Paper" aesthetic and tactical performance logic.

## 🚀 Quick Start
1. Ensure **Node.js** and **MongoDB** are operational.
2. Configure your `.env` in the `backend/` folder (Refer to [requirements.md](./requirements.md)).
3. Run `npm install` in both `frontend` and `backend` sectors.
4. Launch the Hub using `npm run dev` in both directories.

## ⚡ Core Features
- **Tactical Ranking System**: Live rank calculation based on performance averages.
- **Academic Hub**: Peer-to-peer note sharing with high-fidelity HTML downloads.
- **Announcement Stream**: Direct broadcasts from Faculty to students.
- **Study Timer**: Session-based activity tracking.
- **Premium UI**: Ultra-clean, pro-genius aesthetic with custom notification modules.

## 🛰️ Technical Architecture
- **Frontend**: React 19, Vite, Tailwind CSS 4, Framer Motion.
- **Backend**: Node.js, Express 5, MongoDB (Mongoose), JWT Auth.
